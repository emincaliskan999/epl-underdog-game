# EPL Underdog Guess Game

A simple static web app for your 1XBET content idea.

## Files
- `index.html` → page structure
- `styles.css` → design
- `script.js` → game logic and match data

## Edit teams / odds
Open `script.js` and update the `matches` array.

## Run locally
Just open `index.html` in your browser.

## Deploy to Render
1. Create a GitHub repo.
2. Upload these files.
3. In Render, create a **Static Site**.
4. Connect the GitHub repo.
5. Use these settings:
   - Build Command: leave empty
   - Publish Directory: `.`
6. Deploy.
